# Codeapp - SCM Financial Management Plan

This repository contains the architecture and Supabase schema plan for a SCM financial management tool with three user profiles (Administrateur, Médecin, Remplaçant). Use `docs/application_architecture.md` for the functional and technical overview and `docs/supabase_schema.sql` to create the database schema, RLS policies, and helper triggers in Supabase.
